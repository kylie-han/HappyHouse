# HappyHouse Project
## 🏘프로젝트 내용 및 구성

해피하우스는 이사를 하고자 하는 동네에서 원하는 주택을 찾아낼 목적으로 매매/전, 월세에 따른 아파트/연립다세대별 거래정보를 조회할 수 있는 웹서비스 입니다.

## 프로젝트 기능

 - 거래정보 조회
 - 구글맵에 위치정보 표시
 - 공지사항, 질문게시판을 이용한 사용자와 관리자간의 커뮤니케이션
 - 페이징 처리
